# Personal Website / Online CV

This is my personal portfolio website built as a static web project. It serves as my digital identity for future job applications and showcases my background, skills, and projects.

## 🔗 Live Demo

[View the website](https://QianLi071.github.io/my-profolio/)

## 📁 Project Structure
├── index.html # Home page
├── about.html # About me (education, skills, interests)
├── projects.html # Projects gallery
├── contact.html # Contact form & info
├── style.css # Global styles (responsive)
└── images/ # Profile picture and project screenshots

text

## 🛠️ Tech Stack

- **HTML5** (semantic tags)
- **CSS3** (Flexbox & Grid, responsive design)
- No frameworks — pure HTML/CSS

## 📄 Pages

- **Home**: Hero section, skills overview, featured projects preview
- **About**: Introduction, education, technical skills, interests
- **Projects**: Detailed cards for my key projects
- **Contact**: Contact information and a static contact form

## 📱 Responsive Design

The website is fully responsive and works seamlessly on desktop, tablet, and mobile devices.
